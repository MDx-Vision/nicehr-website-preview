# Claude Code Project Files

This directory contains Claude Code configuration and session history.

## Structure

```
.claude/
├── sessions/     # Exported conversation transcripts
└── README.md     # This file
```

## Exporting Sessions

To save a conversation transcript:

```
/export .claude/sessions/YYYY-MM-DD-description.md
```

## Project Memory

The main project memory file is `CLAUDE.md` in the project root.
