# Hero Images

Place full-width hero banner images here.

## Specifications
- **Dimensions:** 1920x1080px minimum
- **Format:** JPG, 80% quality
- **File size:** Under 300KB after compression

## Naming Convention
```
hero-[page]-[description].jpg
```

## Needed Images
- [ ] `hero-homepage-team-meeting.jpg` - Replace Unsplash stock
- [ ] `hero-about-team-group.jpg` - Team photo
- [ ] `hero-services-planning.jpg` - Strategy session
- [ ] `hero-why-tng-consultants.jpg` - Consultants at work
- [ ] `hero-contact-office.jpg` - Office exterior or reception

## Tips
- Images get a dark navy overlay (70-90% opacity)
- Mid-tone images work better than very bright or dark
- Horizontal compositions only
- Avoid text in images (overlay covers it anyway)
